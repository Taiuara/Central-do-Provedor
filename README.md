# Central do Provedor - PingDesk

Sistema de gestão para provedores de internet desenvolvido em React + Next.js.  
Aqui, donos de provedores acessam dashboards com dados de atendimento, vendas e chamados, de forma simples e funcional.

## 🔧 Tecnologias usadas

- React.js + Next.js
- TailwindCSS
- Google Sheets API
- JavaScript (front-end)
- Google Apps Script (back-end)
